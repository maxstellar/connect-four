# connect-four

the beginning of a connect 4 game!

written in Lua and its LOVE2D framework, this project is a simple engine for connect 4 games with a clean and simple UI.
it is written in a lightweight manner in the hopes that people (me?) will be able to add more features, such as powerups or other cool gimmicks!
this project was a beginner project intended to help me learn LOVE2D!
